# Reasoning & Inference

Standard AI memory relies on probabilities. If you embed chat logs, an LLM
retrieves chunks and _guesses_ the answer. Worlds operates on a **Neuro-Symbolic
Architecture**. It uses LLMs to understand language, but stores that
understanding as **Symbolic Logic** in a graph.

By building logical frameworks on top of neural outputs, the engine performs
**Verifiable Inference**—discovering facts that were never explicitly stated.

## Finding Hidden Relationships

Because knowledge is stored as connected Items and Triples, Worlds excels at
graph traversal.

**The Scenario:**

1. _Source A:_ "Alice manages the Frontend Team."
2. _Source B:_ "Bob committed code for the Frontend Team."

When asked _"Who does Bob report to?"_, a standard vector search might fail if
Alice and Bob were never mentioned together. Worlds simply traverses the graph:
`Bob` -> `belongsTo` -> `Frontend Team` -> `managedBy` -> `Alice`. The agent
infers the relationship deterministically.

## Resolving Contradictions

Reasoning is essential for maintaining memory integrity. When facts contradict,
logic dictates the outcome.

If a user says _"My favorite color is Blue"_ and later says _"My favorite color
is Red"_, the engine analyzes the new triple against the existing state. It can
formally override the old fact, ensuring the agent always has a single "ground
truth" rather than two conflicting text chunks.

## Meta-Learning

Unlike neural models that require retraining to learn new rules, Worlds supports
**Meta-Learning**. You can update the "reasoning layers" instantly. If you teach
the system a new logical rule, it applies to all future interactions without a
single training run.

### Power Users: SPARQL

This precision is powered by **SPARQL**, the query language for the Semantic
Web. While the SDK abstracts this, you can execute raw SPARQL for complex
logical traversal.

```sparql
PREFIX w: <https://wazoo.dev/schema#>
SELECT ?project
WHERE {
  ?user w:name "Alice" ;
        w:managerOf ?team .
  ?team w:assignedTo ?project .
}
```
