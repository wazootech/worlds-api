Worlds is a **Neuro-Symbolic Memory API** designed for the AI era. It provides a
stateful infrastructure layer that bridges the gap between the probabilistic
nature of LLMs and the deterministic precision of structured data.

## Core Philosophy

At its heart, Worlds enables agents to maintain a **persistent, queryable world
model**. Unlike traditional vector databases that rely solely on similarity,
Worlds allows for relational logic and symbolic reasoning.

### The Memory Hierarchy

Worlds implements an OS-inspired memory hierarchy for agents:

- **Core Memory**: Immediate, high-priority context like active goals or user
  profiles.
- **Recall Memory**: Searchable history of past interactions and state changes
  across sessions.
- **Archival Memory**: Vast structured knowledge bases built from files, code,
  and external data.
