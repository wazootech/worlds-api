## Overview

**Worlds API** is a managed neuro-symbolic infrastructure layer for querying and reasoning over [SPARQL](https://www.w3.org/TR/sparql11-overview/)-compatible knowledge bases at the edge. Place malleable, high-precision knowledge within arm's reach of your **agentic workflows** and **autonomous assistants**.

Operate on the concept of **Worlds**â€”isolated, queryable knowledge bases for your agents.

## Modular design

The Worlds Platform supports everything from **local-first** development to **cloud-native** production.

- **Local-First**: Run the API Server and Console entirely on your machine.
- **Hybrid**: Mix and match components, like a local Console with remote Worlds APIs deployed on Deno Deploy.
- **Cloud-Native**: Deploy to the edge with Deno Deploy and Turso for global intelligence.
