---
title: "Update world"
api: "PUT https://api.wazoo.dev/v1/worlds/{worldId}"
description: "Update an existing world."
---

## Path Parameters

<ParamField path="worldId" type="string" required>
  The unique identifier of the world.
</ParamField>

## Body Parameters

<ParamField body="name" type="string">
  The new name of the world.
</ParamField>

<ParamField body="description" type="string">
  The new description of the world.
</ParamField>

## Response

The endpoint returns a 200 OK status on success.
