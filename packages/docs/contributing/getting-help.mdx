---
title: "Getting Help"
---

If you need help or have questions about the Worlds Platform, please use the
following channels.

## Support channels

### GitHub Issues

For technical bugs, feature requests, and documentation improvements, please
open an issue on the
[GitHub repository](https://github.com/wazootech/worlds/issues). This is the
best way to track progress and engage with the maintainers.

### Discord

Join the community on **Discord**, which is coming soon, to chat with other
developers, share your projects, and get real-time support.

### Email

For sensitive issues or commercial inquiries, you can reach us at
[support@wazoo.dev](mailto:support@wazoo.dev).

## Before you ask

To get the fastest help:

1.  **Check the docs**: Search the documentation for your topic.
2.  **Search issues**: Look for similar problems on GitHub.
3.  **Check the FAQ**: Review common questions in the tutorials and guides.
