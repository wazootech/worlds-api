The **Worlds API SDK** is the easiest way to interact with the API. It provides a fully typed interface for managing worlds, executing SPARQL queries, and performing vector searches.

## Installation

Install the package via your preferred package manager (using JSR):

```bash
npx jsr add @wazoo/worlds-sdk
```

## Basic Usage

Initialize the client with your API key.

```typescript
import { WorldsSdk } from "@wazoo/worlds-sdk";

// Initialize the client
const sdk = new WorldsSdk({
  baseUrl: "https://api.wazoo.dev/v1", // Adjust if using a different endpoint
  apiKey: "your-api-key",
});

// Run a SPARQL query (requires the world ID)
const result = await sdk.worlds.sparql(
  "00000000-0000-0000-0000-000000000000",
  `
  SELECT ?s ?p ?o WHERE {
    ?s ?p ?o .
  } LIMIT 10
`,
);

console.log(result);
```
