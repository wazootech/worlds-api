## Prerequisites

Requires a Worlds API key. Worlds you create can be viewed on the
[Worlds Console](https://console.wazoo.dev).

## Getting Started

Follow these steps to integrate Worlds into your AI application.

<Steps>
  <Step title="Install the SDK">
    Choose your preferred environment to get started with the Worlds SDK.

    <CodeGroup>
      ```bash npm
      npm install @wazoo/worlds-sdk
      ```

      ```bash yarn
      yarn add @wazoo/worlds-sdk
      ```

      ```bash pnpm
      pnpm add @wazoo/worlds-sdk
      ```
    </CodeGroup>

  </Step>

  <Step title="Initialize your first World">
    Create a connection to the Worlds API and initialize your first persistent
    context.

    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        import { WorldsSdk } from "@wazoo/worlds-sdk";

        const sdk = new WorldsSdk({
          apiKey: "YOUR_API_KEY",
        });

        const world = await sdk.worlds.create({
          slug: "my-first-world",
          label: "My First World",
        });
        ```
      </Tab>
      <Tab title="cURL">
        ```bash
        curl -X POST https://api.wazoo.dev/worlds \
          -H "Authorization: Bearer YOUR_API_KEY" \
          -H "Content-Type: application/json" \
          -d '{
            "slug": "my-first-world",
            "label": "My First World"
          }'
        ```
      </Tab>
    </Tabs>

  </Step>

  <Step title="Add context">
    Start feeding your AI agents with persistent, searchable memory.

    <Tabs>
      <Tab title="TypeScript">
        ```typescript
        await sdk.worlds.import("my-first-world", `
          <#memo> <http://purl.org/dc/terms/description> "The secret ingredient is always more compute." .
        `, { format: "turtle" });
        ```
      </Tab>
      <Tab title="cURL">
        ```bash
        curl -X POST https://api.wazoo.dev/worlds/my-first-world/sparql \
          -H "Authorization: Bearer YOUR_API_KEY" \
          -H "Content-Type: application/sparql-update" \
          -d 'INSERT DATA {
            <#memo> <http://purl.org/dc/terms/description> "The secret ingredient is always more compute." .
          }'
        ```
      </Tab>
    </Tabs>

  </Step>
</Steps>
