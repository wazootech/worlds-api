---
title: "Rate limits"
description: "Information about rate limiting on the Worlds platform."
---

To ensure system stability and fair usage, the Worlds API enforces rate limits
 per service account using a token bucket
algorithm. Admin requests are exempt from these limits.

## Mechanism

- **Period:** 1 minute (60,000 ms).
- **Admin:** Exempt.
- **Service account:** Limited per account.
- **Unauthenticated:** Rejected (401).
