# SDK overview

The **Worlds API SDK** provides a fully typed interface for managing worlds,
executing SPARQL queries, and performing vector searches.

## Installation

Install the SDK via JSR:

```bash
npx jsr add @wazoo/worlds-sdk
```

## Basic usage

Initialize the client with your API key.

```typescript
import { WorldsSdk } from "@wazoo/worlds-sdk";

const sdk = new WorldsSdk({
  baseUrl: "https://api.wazoo.dev",
  apiKey: "your-api-key",
});

// Run a SPARQL query
const result = await sdk.worlds.sparql(
  "WORLD_ID",
  `
  SELECT ?s ?p ?o WHERE {
    ?s ?p ?o .
  } LIMIT 10
`,
);

console.log(result);
```
