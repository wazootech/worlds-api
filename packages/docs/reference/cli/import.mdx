---
title: "import"
description: "Bulk digest RDF triples into an existing World."
---

Streams a local file containing RDF assertions into a remote world.

<ParamField path="worldId" type="string" required>
  The unique system identifier of the target world.
</ParamField>

<ParamField path="file_path" type="string" required>
  The relative or absolute local file path to the RDF file.
</ParamField>

<ParamField path="--format" type="string">
  The RDF serialization format. Accepted values include `turtle` and `n-quads`.
  *Alias:* `-f`
</ParamField>

### Usage

```bash
worlds import <worldId> ./data.ttl --format turtle
```
