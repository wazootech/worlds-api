The Worlds API is fully described using the OpenAPI 3.0 specification. You can use this file to generate client SDKs, explore the API with tools like Postman or Insomnia, or integrate with AI agents.

<Card title="Download openapi.yaml" icon="file-arrow-down" href="/openapi.yaml">
  Download the raw YAML specification file.
</Card>

## Usage

You can also access the specification directly via URL:

```bash
https://docs.wazoo.dev/openapi.yaml
```

For most use cases, we recommend using our [official TypeScript SDK](/api-integrations/sdk-overview) which provides a type-safe interface over this specification.
