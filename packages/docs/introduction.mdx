description: "World models as a Service. The open-source context engine for AI."
---

Worlds is a REST API designed to manage, query, update, and reason over
interoperable knowledge bases at the edge. It provides an interoperable context
graph to natively improve agent reasoning without relying on probabilistic
guessing.
