title: "Introduction"
description: "The open-source context engine for AI"
---

Worlds is an open-source context engine designed to manage, query, update, and
reason over interoperable knowledge bases at the edge. At its core, Worlds is an
RDF store in sync with a search index where every triple is indexed. It provides
agents with a deterministic, graph-based memory system that continually improves
output quality and consistency.
