Master the primitives of the neuro-symbolic Semantic Web.

> [!NOTE] **Architectural onboarding: The symbolic environment** Worlds provides
> the environment layer for AI agents. While LLMs process text in isolation,
> Worlds enables them to interact with a stateful, structured memory that
> persists beyond a single prompt.

Worlds increases environment intelligence rather than model size. This
curriculum explains how to build and manage these stateful ecosystems.

## Foundational priming

Internalize the symbolic grammar of the ecosystem before accessing the global
graph. This onboarding sequence isolates core interaction patterns to ensure
cognitive focus remains on the technical curriculum.

<Steps>
  <Step title="Initialize identity">
    In the Worlds ecosystem, every interaction is an item in the graph. Confirm
    your presence by selecting your primary focus.
  </Step>
  <Step title="Establish input logic">
    Our documentation utilizes **progressive disclosure**. Click the "Explore
    the history" card below to reveal the first layer of the curriculum.
  </Step>
  <Step title="Enter the world">
    Access the technical curriculum once you internalize the interface grammar.
  </Step>
</Steps>

## Key takeaways

- **Environment intelligence**: Worlds prioritizes the container over the agent.
- **Modular pedagogy**: Lessons are standalone modules in a logical progression.
- **Socratic friction**: Expect logic puzzles and architectural stress tests.
