Welcome to the **Worlds Academy**.

Worlds focuses on increasing environment intelligence rather than model size.
This curriculum prepares you to build stateful, self-improving AI ecosystems.

## Key takeaways

- Worlds prioritizes environment intelligence over raw model scale.
- The Academy provides modular lessons on neuro-symbolic infrastructure.
- Each lesson functions as a standalone module within a logical progression.

## The curriculum

The Academy organizes lessons into modular chapters. Each chapter covers a
fundamental pillar of Worlds infrastructure.

## How to learn

Lessons follow a logical progression but remain standalone. We recommend
following chapters in order if you are new to RDF and neuro-symbolic AI.
