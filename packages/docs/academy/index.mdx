Welcome to the Worlds Academy. This curriculum guides you from the fundamentals
of world models to building advanced, stateful AI agents.

<Note>
  **Core objective**

By the end of this path, you will understand how to bridge Large Language Models
with structured knowledge graphs, giving your agents persistent, stateful
memory.

</Note>

## Learning path

Follow these steps in order to build your foundation in the Worlds Ecosystem.

<Steps>
  <Step title="World memory fundamentals">
    Understand the concept of world memory and how it differs from traditional RAG. Learn why stateful environments are essential for intelligent agents.

    [Start module](/academy/memory)

  </Step>

  <Step title="Symbolic graph architecture">
    Learn how to structure knowledge using Subject-Predicate-Object triples. Master the atomic unit of knowledge in the Worlds graph.

    [Start module](/academy/graphs)

  </Step>

  <Step title="Logical reasoning with SPARQL">
    Master the basics of querying your world memory with symbolic logic. Learn to extract exact facts and follow logical relationships.

    [Start module](/academy/sparql)

  </Step>

  <Step title="Neuro-symbolic agent integration">
    Connect your AI agent to its memory using neuro-symbolic tool-calling. Build an agent that can read and write to its own world model.

    [Start module](/academy/agents)

  </Step>
</Steps>
