## Get started

Create your first World and run a query in under 5 minutes.

<Steps>
  <Step title="Generate credentials">
    You'll need an API key to authenticate requests and a World ID to store your data. Create these in the [dashboard](https://console.wazoo.dev).
  </Step>

  <Step title="Install the SDK">
    Add the official TypeScript SDK to your project via JSR.

<CodeGroup>

```bash npm
npx jsr add @wazoo/worlds-sdk
```

```bash deno
deno add jsr:@wazoo/worlds-sdk
```

</CodeGroup>

  </Step>

  <Step title="Initialize the client">
    Configure the SDK with your API key to start interacting with the API.

```typescript
import { WorldsSdk } from "@wazoo/worlds-sdk";

const sdk = new WorldsSdk({
  baseUrl: "https://api.wazoo.dev/v1", // Adjust if using a different endpoint
  apiKey: "YOUR_API_KEY",
});

const worldId = "YOUR_WORLD_ID";
```

  </Step>

  <Step title="Add knowledge">
    Insert triples into your World using SPARQL. This example adds a person and their job title.

```typescript
await sdk.worlds.sparql(
  worldId,
  `
  INSERT DATA {
    <http://example.org/ethan> <http://schema.org/name> "Ethan" .
    <http://example.org/ethan> <http://schema.org/jobTitle> "Software Engineer" .
  }
`,
);
```

  </Step>

  <Step title="Query knowledge">
    Retrieve the data you just added to verify the insertion was successful.

```typescript
const result = await sdk.worlds.sparql(
  worldId,
  `
  SELECT ?name ?title WHERE {
    ?person <http://schema.org/name> ?name ;
            <http://schema.org/jobTitle> ?title .
  }
`,
);

console.log(result);
```

    <Check>
      Verify the `result` contains "Ethan" and "Software Engineer".
    </Check>

  </Step>

  <Step title="Search knowledge">
    Perform a semantic search to find information based on meaning rather than exact keywords.

```typescript
const searchResults = await sdk.worlds.search(
  worldId,
  "Who is the software engineer?",
);
console.log(searchResults);
```

  </Step>
</Steps>
