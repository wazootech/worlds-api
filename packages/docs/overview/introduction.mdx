Worlds is a neuro-symbolic memory interface for the AI era. It provides a
stateful infrastructure layer that bridges the gap between the probabilistic
nature of LLMs and the deterministic precision of structured data.

The context infrastructure enables operators to build agents with persistent,
verifiable world models, ensuring reliable recall and auditable reasoning.

## Core philosophy

Worlds bridges the gap between AI intuition and deterministic ownership using
three core pillars.

### Neuro-symbolic synthesis

Worlds uses a hybrid architecture to solve the reasoning gap in modern AI.

- **Neural (LLM intuition)**: Generative AI extracts meaning from unstructured
  data.
- **Symbolic (RDF logic)**: The queryable graph stores that meaning as
  deterministic facts. This ensures your agent navigates a structured map of
  reality rather than guessing context based on probabilities.

### Edge-first autonomy

An agent's memory must live close to the user. Worlds ensures that intelligence
and knowledge live on your local machine, a private server, or at the network
edge.

- **Local autonomy**: Agents never depend on a central cloud for their memory.
- **Low latency**: Local nodes retrieve memory significantly faster than global
  databases.

### The local-first manifesto

As agents become more autonomous, humans must retain ownership of the knowledge
those agents carry.

- **Residency**: Data lives close to you.
- **Ownership**: You own the triples; Worlds acts as the facilitator.
- **Portability**: The graph uses an open standard (RDF), preventing proprietary
  silos.
