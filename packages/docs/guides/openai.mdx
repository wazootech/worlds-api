  **Experimental Integration** This integration guide is a stub. The official
  Worlds capabilities for this framework are under active development. Expect
  API friction and missing language parity.
</Warning>

The [OpenAI SDK](https://platform.openai.com/) acts as the core foundational
access layer for programmatically interfacing with models like GPT-4o and o1.

The [OpenAI Agents SDK](https://platform.openai.com/docs/agents) handles the
stateful orchestration of multi-turn messaging, tool calling, and active thread
tracking.
