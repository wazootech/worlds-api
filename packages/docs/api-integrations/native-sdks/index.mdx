## Feature Highlights

### Type-Safe SPARQL

The SDK handles the plumbing of SPARQL requests, including authentication and
response parsing, while keeping your queries type-safe.

```typescript
const result = await sdk.worlds.sparql(
  worldId,
  `
  SELECT ?name WHERE {
    ?person <http://schema.org/name> ?name .
  }
`,
);
```

### Hybrid Search

Combine vector similarity with full-text search. This is ideal for finding
information when you only have a vague concept or a natural language question.

```typescript
const searchResults = await sdk.worlds.search(worldId, "conceptual question");
```

### AI Tool-Calling

The SDK exposes your Worlds as tools that adhere to the standard
[AI SDK tool interface](https://ai-sdk.dev/docs/foundations/tools), making them
directly compatible with various LLM orchestrators (like Vercel AI SDK,
LangChain, and others) to allow agents to "read and write" to the graph.

## Configuration Options

When initializing the client, you can customize retry logic, custom headers, and
timeout behaviors:

```typescript
const sdk = new WorldsSdk({
  baseUrl: "https://api.wazoo.dev",
  apiKey: "your-api-key",
  // Advanced configuration
  fetch: customFetchFn,
});
```
