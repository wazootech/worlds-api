# Welcome to Worlds

Worlds is a **Neuro-Symbolic Memory API** designed for the AI era. It provides a
stateful infrastructure layer that bridges the gap between the probabilistic
nature of LLMs and the deterministic precision of structured data.

## Core Philosophy

At its heart, Worlds enables agents to maintain a **persistent, verifiable world
model**. Unlike traditional vector databases that rely solely on similarity,
Worlds allows for relational logic and symbolic reasoning.
